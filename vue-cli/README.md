# Project

* **Git Init**

```shell
git clone ssh://git@git.elemenx.com:2233/eshop/b2c/fe.git
git submodule init
git submodule update
```

or

```shell
git clone --recursive ssh://git@git.elemenx.com:2233/eshop/b2c/fe.git
git submodule update
```

* **Development:**

```shell
npm start
```

* **Build:**

```shell
npm run build
```

* **Update Vendor:**

```shell
npm run dll
```
